# 📖 00-学习准备

> 在开始学习 LangGraph 之前，确保你的环境已正确配置

## 🎯 本章目标

完成本章后，你将：
- ✅ 确认具备学习 LangGraph 的前置知识
- ✅ 完成 Python 环境配置
- ✅ 安装所有必需的依赖库
- ✅ 配置好 API Key
- ✅ 成功运行第一个验证脚本

## 📚 前置知识检查

在开始学习 LangGraph 之前，建议具备以下基础知识：

### 必需知识
- [ ] **Python 基础**：变量、函数、类、模块
- [ ] **Python 3.10+**：类型注解、async/await
- [ ] **包管理**：pip 安装和管理依赖

### 推荐知识
- [ ] **LangChain 基础**：了解 LangChain 的基本概念
- [ ] **异步编程**：async/await 基本用法
- [ ] **API 调用**：了解 HTTP API 的基本概念

### 加分项
- [ ] **向量数据库**：了解 Chroma、FAISS 等
- [ ] **LLM 基础**：了解大语言模型的基本原理
- [ ] **Agent 概念**：了解 Agent 的基本架构

**如果没有完全掌握？** 不要担心！边学边练是最好的方式。遇到不懂的概念可以回头查阅。
