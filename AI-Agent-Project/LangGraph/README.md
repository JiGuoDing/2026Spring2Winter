# LangGraph 完全学习指南

> 从零基础上手到独立开发 LangGraph Agent 的一站式学习项目

## 🎯 项目定位

本项目是一个**系统化的 LangGraph 学习教程**，专为希望掌握 LangGraph 框架进行 Agent 开发的开发者设计。通过本教程，你将：

- ✅ 理解 LangGraph 的核心概念与架构设计
- ✅ 掌握 State、Node、Edge、Graph 的使用方法
- ✅ 能够独立开发完整的 LangGraph Agent
- ✅ 学会状态管理、记忆系统、RAG 集成等高级技巧
- ✅ 了解生产环境最佳实践和性能优化方法

## 🚀 快速开始

### 1. 安装依赖
```bash
cd LangGraph
pip install -r requirements.txt
```

### 2. 配置环境
```bash
cp .env.example .env
# 编辑 .env 文件，填入你的 API Key
```

### 3. 运行第一个示例
```bash
python "01-LangGraph 核心概念/examples/basic_state.py"
```

## 📚 前置要求

- **Python 基础**：Python 3.10+，了解基本语法和异步编程
- **LangChain 基础**：了解 LangChain 的基本概念（推荐先学习 openai-tutorial）
- **API Key**：OpenAI 或通义千问的 API Key
- **开发环境**：VS Code 或 PyCharm，推荐安装 Python 插件

## 🎓 学习建议

1. **循序渐进**：按照 00→08 的顺序学习，不要跳过基础章节
2. **动手实践**：每个示例都要亲自运行，尝试修改代码观察效果
3. **记录笔记**：记录关键知识点和遇到的问题
4. **完成练习**：每章末尾的练习题务必完成
