## 📖 使用方法

1. **克隆后首先安装依赖**:
   ```bash
   pip install -r requirements.txt
   ```

2. **配置环境变量**:
   ```bash
   cp .env.example .env
   # 编辑 .env 填入 OPENAI_API_KEY
   ```

3. **从基础开始学习**:
   ```bash
   # 阅读文档
   cat 01-基础入门/README.md
   
   # 运行示例
   python 01-基础入门/hello_world.py
   ```

4. **遇到问题查看 FAQ**:
   ```bash
   cat 06-常见问题/README.md
   ```

## 💡 学习建议

1. **循序渐进**: 按顺序学习，不要跳过基础
2. **动手实践**: 每个示例都要亲自运行
3. **记录笔记**: 记录关键知识点和踩坑经验
4. **扩展练习**: 在示例基础上添加自己的功能
