# 📚 OpenAI 库学习教程

> 从入门到精通的完整学习路径，专为 Agent 开发打造

## 🎯 项目简介

这是一个系统化的 OpenAI Python 库学习教程，包含从基础安装到高级 Agent 开发的完整内容。教程特别针对 Agent 开发场景，提供了丰富的实战示例和最佳实践。

## ✨ 特色亮点

- 🎓 **渐进式学习**: 6 大章节，循序渐进
- 💻 **丰富示例**: 8 个完整可运行示例
- 🔧 **实战导向**: 聚焦 Agent 开发实际需求
- 📦 **工具齐全**: 提供实用工具函数库
- 🛡️ **最佳实践**: 涵盖安全、性能、测试等
- ❓ **FAQ 支持**: 常见问题详细解答

## 🚀 快速开始

### 1. 安装依赖
```bash
cd openai-tutorial
pip install -r requirements.txt
```

### 2. 配置环境
```bash
cp .env.example .env
# 编辑 .env 文件，填入你的 OPENAI_API_KEY
```

### 3. 运行第一个示例
```bash
python 01-基础入门/hello_world.py
```
