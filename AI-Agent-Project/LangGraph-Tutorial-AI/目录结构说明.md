## 文件类型说明

### 文档文件（.md）
- **README.md**：章节概览，快速了解章节内容
- **主题文档**：详细讲解某个概念或技术
- **指南文档**：操作步骤和最佳实践

### 代码文件（.py）
- **示例代码**：演示特定功能的完整代码
- **可运行**：所有示例代码都可以直接运行
- **详细注释**：关键步骤都有中文注释

### 配置文件
- **requirements.txt**：Python 依赖列表
- **.env.example**：环境变量模板
- **.gitignore**：Git 忽略规则

## 如何导航

### 按学习顺序
1. 从 [00-学习准备](00-学习准备/README.md) 开始
2. 按章节顺序：01 → 02 → 03 → ... → 08
3. 每章先读 README，再读主题文档，最后运行示例

### 按主题查找
- **基础概念**：01-LangGraph 核心概念
- **Agent 开发**：02-第一个 LangGraph Agent
- **记忆系统**：03-状态管理与记忆系统
- **高级技巧**：04-高级模式与技巧
- **RAG 集成**：05-RAG 增强 Agent
- **生产部署**：06-生产环境最佳实践
- **实战项目**：07-实战项目
- **问题排查**：08-常见问题与调试

### 按示例代码查找
所有示例代码都在对应章节的 `examples/` 目录下，可以直接运行：
```bash
python "01-LangGraph 核心概念/examples/basic_state.py"
```
