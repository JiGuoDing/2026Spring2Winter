# 📚 LangGraph 学习教程完整索引

> 从零基础上手到独立开发 LangGraph Agent 的完整学习路径

## 🎯 项目简介

这是一个系统化的 LangGraph 学习教程，包含从基础概念到高级实战的完整内容。教程专为 Agent 开发场景设计，提供了丰富的示例代码和最佳实践指导。

- 🎓 **渐进式学习**: 9 大章节，循序渐进

- 🔧 **实战导向**: 3 个完整实战项目
- 📦 **工程规范**: 生产环境最佳实践
- 🛡️ **问题排查**: 详细 FAQ 和调试指南
- 📝 **详尽文档**: 每个步骤都有详细说明

## 🚀 快速开始

### 1. 安装依赖
```bash
cd LangGraph
pip install -r requirements.txt
```

### 2. 配置环境
```bash
cp .env.example .env
# 编辑 .env 文件，填入你的 API Key
```

### 3. 运行第一个示例
```bash
python "01-LangGraph 核心概念/examples/basic_state.py"
```
