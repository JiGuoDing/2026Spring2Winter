## 🚀 下一步行动

### 立即开始

```bash
# 1. 进入项目目录
cd LangGraph

# 2. 安装依赖
pip install -r requirements.txt

# 3. 配置环境
cp .env.example .env
# 编辑 .env 填入 API Key

# 4. 运行第一个示例
python "01-LangGraph 核心概念/examples/basic_state.py"

# 5. 运行完整 Agent
python "02-第一个 LangGraph Agent/examples/first_agent.py"
```
