## Step 1: 安装 Python

### 要求
- **Python 版本**: 3.10 或更高
- **推荐版本**: Python 3.11 或 3.12

### Windows 系统

1. 访问 [Python 官网](https://www.python.org/downloads/)
2. 下载最新版本的 Python 安装程序
3. 运行安装程序，**务必勾选 "Add Python to PATH"**
4. 点击 "Install Now"

验证安装：
```bash
python --version
# 应该显示: Python 3.10.x 或更高
```

### macOS 系统

使用 Homebrew 安装：
```bash
# 安装 Homebrew（如果还没有）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 安装 Python
brew install python

# 验证
python3 --version
```

### Linux 系统 (Ubuntu/Debian)

```bash
sudo apt update
sudo apt install python3 python3-pip python3-venv

# 验证
python3 --version
```

## Step 2: 创建虚拟环境

虚拟环境可以隔离项目依赖，避免与其他项目冲突。

### 使用 venv（推荐）

```bash
# 进入项目目录
cd LangGraph

# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
# Windows:
venv\Scripts\activate

# macOS/Linux:
source venv/bin/activate
```

激活后，命令行前面会显示 `(venv)`。

### 使用 conda

```bash
# 创建 conda 环境
conda create -n langgraph python=3.11

# 激活环境
conda activate langgraph
```

### 验证虚拟环境

```bash
# 查看 Python 路径
which python  # macOS/Linux
where python  # Windows

# 应该指向虚拟环境目录
```

## Step 3: 安装依赖

### 升级 pip

```bash
python -m pip install --upgrade pip
```

### 安装项目依赖

```bash
pip install -r requirements.txt
```

**国内用户加速**：
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 验证安装

```bash
# 检查核心库
python -c "import langgraph; print(f'LangGraph: {langgraph.__version__}')"
python -c "import langchain; print(f'LangChain: {langchain.__version__}')"
python -c "import openai; print(f'OpenAI: {openai.__version__}')"
```

应该看到类似输出：
```
LangGraph: 0.2.x
LangChain: 0.3.x
OpenAI: 1.x.x
```

## Step 4: 配置 API Key

### 选择模型提供商

你可以选择以下任一提供商：

1. **OpenAI**（推荐，功能最全）
   - 官网: https://platform.openai.com
   - 模型: gpt-3.5-turbo, gpt-4
   
2. **通义千问**（国内可用）
   - 官网: https://dashscope.console.aliyun.com
   - 模型: qwen-turbo, qwen-plus

### 获取 API Key

#### OpenAI
1. 注册 [OpenAI 账号](https://platform.openai.com/signup)
2. 访问 [API Keys 页面](https://platform.openai.com/api-keys)
3. 点击 "Create new secret key"
4. 复制并保存你的 API Key

#### 通义千问
1. 注册 [阿里云账号](https://www.aliyun.com)
2. 开通 [DashScope 服务](https://dashscope.console.aliyun.com)
3. 访问 [API-KEY 管理](https://dashscope.console.aliyun.com/apiKey)
4. 创建并复制 API Key

### 配置 .env 文件

```bash
# 复制模板
cp .env.example .env

# 编辑 .env 文件
# Windows:
notepad .env

# macOS/Linux:
nano .env
```

在 `.env` 文件中填入你的 API Key：

```env
# 使用 OpenAI
OPENAI_API_KEY=sk-your-api-key-here

# 或使用通义千问
DASHSCOPE_API_KEY=your-dashscope-key-here
```

**注意**：
- 不要删除引号
- 不要在 Key 前后加空格
- `.env` 文件已被 .gitignore 忽略，不会提交到 Git

## Step 5: 验证环境

### 运行验证脚本

创建测试文件 `test_env.py`：

```python
"""环境验证脚本"""
import os
from dotenv import load_dotenv

# 加载环境变量
load_dotenv()

def test_imports():
    """测试库导入"""
    print("✓ 测试 1: 库导入")
    try:
        import langgraph
        import langchain
        import openai
        print(f"  - LangGraph: {langgraph.__version__}")
        print(f"  - LangChain: {langchain.__version__}")
        print(f"  - OpenAI: {openai.__version__}")
        print("  ✓ 所有库导入成功\n")
        return True
    except ImportError as e:
        print(f"  ✗ 库导入失败: {e}\n")
        return False

def test_api_key():
    """测试 API Key 配置"""
    print("✓ 测试 2: API Key 配置")
    openai_key = os.getenv('OPENAI_API_KEY')
    dashscope_key = os.getenv('DASHSCOPE_API_KEY')
    
    if openai_key and openai_key != 'your-openai-api-key-here':
        print(f"  - OpenAI API Key: 已配置 ({openai_key[:8]}...)")
        return 'openai'
    elif dashscope_key and dashscope_key != 'your-dashscope-api-key-here':
        print(f"  - DashScope API Key: 已配置 ({dashscope_key[:8]}...)")
        return 'dashscope'
    else:
        print("  ✗ 未配置有效的 API Key")
        print("  请编辑 .env 文件，填入你的 API Key\n")
        return None

def test_basic_graph():
    """测试基础图构建"""
    print("✓ 测试 3: LangGraph 基础功能")
    try:
        from langgraph.graph import StateGraph, END
        from typing import TypedDict
        
        class State(TypedDict):
            message: str
        
        def node_1(state: State):
            return {"message": state["message"] + " -> Node 1"}
        
        def node_2(state: State):
            return {"message": state["message"] + " -> Node 2"}
        
        graph = StateGraph(State)
        graph.add_node("node_1", node_1)
        graph.add_node("node_2", node_2)
        graph.add_edge("node_1", "node_2")
        graph.set_entry_point("node_1")
        graph.add_edge("node_2", END)
        
        app = graph.compile()
        result = app.invoke({"message": "Start"})
        print(f"  - 图执行结果: {result['message']}")
        print("  ✓ LangGraph 基础功能正常\n")
        return True
    except Exception as e:
        print(f"  ✗ LangGraph 测试失败: {e}\n")
        return False

def main():
    print("=" * 50)
    print("LangGraph 环境验证")
    print("=" * 50 + "\n")
    
    results = []
    results.append(test_imports())
    api_provider = test_api_key()
    results.append(api_provider is not None)
    results.append(test_basic_graph())
    
    print("=" * 50)
    if all(results):
        print("✓ 所有测试通过！环境配置正确。")
        print(f"✓ 你可以开始学习 LangGraph 了！")
    else:
        print("✗ 部分测试失败，请检查上面的错误信息。")
    print("=" * 50)

if __name__ == "__main__":
    main()
```

运行验证脚本：
```bash
python test_env.py
```

**期望输出**：
```
==================================================
LangGraph 环境验证
==================================================

✓ 测试 1: 库导入
  - LangGraph: 0.2.x
  - LangChain: 0.3.x
  - OpenAI: 1.x.x
  ✓ 所有库导入成功

✓ 测试 2: API Key 配置
  - OpenAI API Key: 已配置 (sk-your...)

✓ 测试 3: LangGraph 基础功能
  - 图执行结果: Start -> Node 1 -> Node 2
  ✓ LangGraph 基础功能正常

==================================================
✓ 所有测试通过！环境配置正确。
✓ 你可以开始学习 LangGraph 了！
==================================================
```

## Step 6: IDE 配置

### VS Code（推荐）

1. **安装 Python 插件**
   - 打开 VS Code
   - 扩展市场搜索 "Python"
   - 安装 Microsoft 的 Python 插件

2. **选择 Python 解释器**
   - `Ctrl+Shift+P` (Windows) 或 `Cmd+Shift+P` (Mac)
   - 输入 "Python: Select Interpreter"
   - 选择虚拟环境中的 Python

3. **推荐插件**
   - Python
   - Pylance
   - Python Environment Manager

### PyCharm

1. **打开项目**
   - File → Open → 选择 LangGraph 目录

2. **配置解释器**
   - Settings → Project → Python Interpreter
   - 选择虚拟环境中的 Python

3. **启用自动格式化**
   - Settings → Tools → Black formatter
   - 启用 "Run on save"

## 常见问题

### Q1: pip 安装速度很慢

**解决方案**：使用国内镜像源
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q2: 提示 "No module named 'langgraph'"

**解决方案**：
1. 确认虚拟环境已激活（命令行前面有 `(venv)`）
2. 重新安装依赖：`pip install -r requirements.txt`
3. 检查 Python 路径：`which python` 或 `where python`

### Q3: .env 文件没有生效

**解决方案**：
1. 确认文件名为 `.env`（不是 `.env.txt`）
2. 确认 `.env` 文件在项目根目录
3. 代码中必须有 `load_dotenv()`
4. 重启 Python 进程

### Q4: Python 版本太低

**解决方案**：
```bash
# 查看当前版本
python --version

# 如果低于 3.10，需要升级
# Windows: 从官网下载安装新版本
# macOS: brew install python@3.11
# Linux: sudo apt install python3.11
```

### Q5: 虚拟环境激活失败

**Windows PowerShell**:
```bash
# 可能遇到执行策略问题
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

**macOS/Linux**:
```bash
# 确认使用 source 命令
source venv/bin/activate
```
